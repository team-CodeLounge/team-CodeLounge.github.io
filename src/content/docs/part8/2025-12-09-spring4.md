## Mysql 컨테이너 접속법
```bash
# 접속 명령어 -> 루트 비밀번호 입력
docker exec -it server-base-mysql mysql -u root -p
```

## 데이터베이스 생성
```sql
CREATE DATABASE server_base_db;
```

## 데이터베이스 삭제
```sql
DROP DATABASE server_base_db;
```

## 데이터베이스 접근
```sql
use server_base_db;
```

## Users 테이블 만들기
```sql
// users
Create Table users (
   id BIGINT NOT NULL AUTO_INCREMENT,
   uid VARCHAR(255) NOT NULL,
   uuid VARCHAR(36) NOT NULL DEFAULT (UUID()),
   provider ENUM('APPLE', 'GOOGLE', 'KAKAO') NOT NULL,
   email VARCHAR(255) NULL,
   nickname VARCHAR(255) NULL,
   role ENUM('ADMIN', 'MEMBER') NOT NULL,
   created_at DATETIME NULL,
   updated_at DATETIME NULL, 
   PRIMARY KEY (id),
   UNIQUE KEY uq_users_uid (uid)
);
```

## 특정 행 삭제
```sql
DELETE FROM users WHERE id = 1;
DELETE FROM users WHERE uid = '102861577172710745698';
```
