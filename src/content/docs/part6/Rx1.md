## ReactiveX
RxSwift를 알기 위해 ReactiveX에 대해 알아야 한다. ReactiveX 혹은 Reactive Extension, 혹은 Rx라고도 부른다. Rx는 마이크로소프트에서 개발한 비동기 이벤트를 핸들링해주는 오픈소스 라이브러리이다.

## Observer 패턴
Observer 패턴이란 관찰자(Observer)가 대상이 되는 객체를 등록 함으로서 해당 객체의 상태 변화를 관찰하고 관찰 대상인 객체가 상태 변화로 이벤트를 발생시킬 때 이 이벤트를 감지하여 처리한다.

## 개념
파이프라인을 만들고 총알들을 하나씩 보낸다고 가정하자.  
요소 - 보내는 것(총알)  
구독 - 파이프라인을 연결한다는 의미   
연산자 - 중간에 요소를 처리한다는 의미  
스레드 - 어떤 스레드에서 사용할 지 설정 가능
