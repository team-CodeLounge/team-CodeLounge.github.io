## 연산 규칙
- 같은 타입의 데이터끼리만 연산 및 저장이 가능합니다.

## 연산자 종류
1. 연산자 개수에 따라 구분합니다.
- 단항연산자
- 이항연산자
- 삼항연산자
```swift
연산자
 ├─ 단항 연산자
 │   └─ +a, -a
 │
 ├─ 이항 연산자
 │   ├─ 산술 연산자        (+, -, *, /, %)
 │   ├─ 비교 연산자        (==, <, >, <=, >=)
 │   ├─ 논리 연산자        (&&, ||)
 │   ├─ 비트 연산자        (&, |, ^, <<, >>)
 │   └─ 복합 할당 연산자    (+=, -=, *=, /=, %=)
 │
 └─ 삼항 연산자
     └─ 조건 연산자 ( ? : )
```

## 할당 연산자(Assignment)
- 오른쪽 값을 왼쪽에 대입(=할당)한다는 의미입니다.
```swift
let age = 26 + 1
```

## 산술 연산자(Arithmetic)
- Swift에 내장된 수학적인 연산자입니다.
```swift
let a = 10
let b = 3

a + b   // 덧셈   → 13
a - b   // 뺄셈   → 7
a * b   // 곱셈   → 30
a / b   // 나눗셈  → 3 (정수 나눗셈)
a % b   // 나머지  → 1

// 모듈러 예시)
// Index가 게임을 320분 했다면 5시간 20분동안 게임을 하였음.
320 / 60 => 5(몫)
320 % 60 => 20(나머지)
```

## 복합 할당 연산자
- 산술 연산과 할당을 동시에 수행하는 연산자입니다.
- 변수 = 변수 연산 값 형태를 간단하게 표현할 수 있습니다.
- 왼쪽 피연산자는 반드시 var여야 합니다.
```swift
var a = 10

a += 3   // 덧셈 후 할당 → a = a + 3  → 13
a -= 2   // 뺄셈 후 할당 → a = a - 2  → 11
a *= 4   // 곱셈 후 할당 → a = a * 4  → 44
a /= 5   // 나눗셈 후 할당 → a = a / 5 → 8 (정수 나눗셈)
a %= 3   // 나머지 후 할당 → a = a % 3 → 2
```

## 비교 연산자(Comparison)
```swift
let a = 10
let b = 3

a == b   // 같음         → false
a != b   // 다름         → true
a > b    // 큼           → true
a < b    // 작음         → false
a >= b   // 크거나 같음  → true
a <= b   // 작거나 같음  → false
```

## 논리 연산자(Logical)
- Bool 값끼리의 논리 연산을 수행합니다.
```swift
let isLogin = true
let isAdult = false

isLogin && isAdult   // AND (둘 다 true) → false
isLogin || isAdult   // OR  (하나라도 true) → true
!isLogin             // NOT (부정) → false
```

## 비트 연산자(Bitwise)
- 정수 값을 2진수 비트 단위로 연산합니다.
- 시스템, 성능 최적화, 플래그 처리에서 자주 사용됩니다.
```swift
let x: UInt8 = 0b1010  // 10
let y: UInt8 = 0b1100  // 12

x & y   // AND  → 0b1000 (8)
x | y   // OR   → 0b1110 (14)
x ^ y   // XOR  → 0b0110 (6)
~x      // NOT  → 0b0101 (245, UInt8 기준)

// 시프트 연산자 1)
let shift = UInt8(1)
shift << 3   // 왼쪽 시프트 → 0b00001000 (8)
shift >> 1   // 오른쪽 시프트 → 0b00000000 (0)

// 시프트 연산자 2)
let x = 8
x << 1   // 16  (×2)
x >> 1   // 4   (÷2)

// 비트 연산 활용 예시(플래그 처리)
let READ  = 0b001
let WRITE = 0b010
let EXEC  = 0b100

var permission = READ | WRITE   // 읽기 + 쓰기

permission & READ != 0   // 읽기 권한 존재 → true
permission & EXEC != 0   // 실행 권한 없음 → false
```
