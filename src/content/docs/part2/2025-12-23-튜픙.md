## 튜플
- 연관된 데이터의 조합으로 어떤 형태든 만들 수 있는 타입입니다.
- 변수 선언과 동시에 해당 멤버의 종류와 개수는 결정되므로 초가 및 삭제바 불가능합니다.
- 리턴값으로 여러 값을 반환할 수 있습니다.

## Named Tuple
- 이름이 매겨진 튜플이라고도 부르며 데이터 각 요소에 이름을 붙일 수 있습니다.
```swift
let iOS = ("Swift", 6.0)
let namediOS = (language: "Swift", version: 6.0)

print(iOS.0, iOS.1)
print(namediOS.language, namediOS.version)
print(namediOS.0, namediOS.1)
```

## 튜플 변수/상수 바인딩
- 튜플 각 요소의 데이터를 쉽게 바인딩 할 수 있습니다.
```swift
let (name, age, grade) = ("Index", 26, 4.5)
print(name)
print(age)
print(grade)
```

## 예시
```swift
let iOS = ("Swift", 6.0)

// if문 예시
if iOS.0 == "Swift" && iOS.1 == 6.0 {
    print("Swift 6.0 입니다.")
}

if iOS == ("Swift", 6.0) {
    print("Swift 6.0 입니다.")
}

// switch문 예시1)
switch iOS {
case ("Swift", 4.0):
    print("Swift 버전 4.0입니다.")
case ("Swift", 5.0): 
    print("Swift 버전 5.0입니다.")
case ("Swift", 6.0): 
    print("Swift 버전 6.0입니다.")
default: 
    break
}

// switch문 예시2)
typealias Point = (Int, Int)
let point = Point(1, 2)

switch point {
case (let x, let y):
    print("좌표는 \(x), \(y) 입니다.")
}

// switch문 예시3)
switch point {
case let (x, y) where x == y:
    print("좌표가 일치합니다: (\(x), \(y)")
default:
    break
}
```
