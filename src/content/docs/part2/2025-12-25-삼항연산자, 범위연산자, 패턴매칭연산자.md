## 삼항연산자
- 세 가지 항을 가진 연산자로 조건에 따라 선택이 두가지인경우, 그리고 리터럴 값을 변수에 대입할 때 주로 사용합니다.
- 삼항 연산자 단독으로 쓰이지는 않고 주로 결과 값을 대입할 때(modifier) 사용됩니다.
- if문과 동일하게 쓸 수 있다.
```swift
// 삼항연산자
var name = a > 19 ? "성인" : "미성년자"

// 조건문
if age > 19 {
    name =  "성인"
} else {
    name = "미성년자"
}
```

## 범위연산자
- 제한된 숫자의 범위를 표기하기 위한 연산자입니다.
- 내림차순 형식으로 표기할 수 없습니다.(하지만 내림차순 형식으로 변환은 가능)
- 실수 형식의 범위도 가능합니다.(잘 사용 안함)
```swift
let a = 1...10  // ClosedRange<Int>
let b = 1...    // PartialRangeFrom<Int>
let c = ...10   // PartialRangeThrough<Int>

let d = 1..<10  // Range<Int>
let e = ..<10   // PartialRangeUpTo<Int>
```

### 범위 연산자 사용 예시
```swift
// 1) switch 문에서 패턴 매칭에 사용한다
let score = 70

switch score {
case 90...100:
    print("A")
case 80..<90:
    print("B")
case 70..<80:
    print("B")
default:
    print("F")
}

// 2) for문과 주로 사용한다
for i in 0..<10 {
    print(i)
}

// 3. 배열의 서브스크립트 문법과 함께 사용한다
let arr = [1, 2, 3, 4, 5]
print(arr[1...2]) // [2, 3]
```

## 패턴매칭 연산자(~=)
- 범위 연산자와 함꼐 쓰이는 연산자로 오른쪽 표현식이 왼쪽의 범위에 포함되는 지에 따라 참과 거짓을 리턴합니다.
```swift
var age: Int = 26
print(10...20 ~= age)   // false
print(20...30 ~= age)   // true
```
