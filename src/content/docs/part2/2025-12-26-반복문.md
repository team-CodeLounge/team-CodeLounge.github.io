## for문
- 범위나 컬렉션에서 item을 하나씩 뽑아 중괄호 내에서 순차적으로 반복 실행함을 의미합니다.
```swift
for item in 범위 or 컬렉션 {
    print(item)
}
```

### for문의 반복상수
- 아래 item 변수가 아닌 상수로 정의되는 이유
    - 반복 대상의 값을 복사해서 전달하기 때문
    - 실수로 반복 로직을 망가뜨리는걸 방지
    - swift철학흔 가능하면 let, 꼭 필요할때만 var를 사용하라는 철학이 담김
```swift
/**
범위에서 값이 하나씩 item에 복사된다
let item: Int = 0
let item: Int = 1
let item: Int = 2
*/
for item in 0..<3 {
    print(item)
}
```

### 주의점
- for 문에서 선언하는 변수는 for문 내에서만 사용 가능합니다.
- 중괄호 내에서 외부의 변수는 접근 가능하지만 외부에서 중괄호 내의 변수는 접근할 수 없습니다.

### for문에서 주로 사용되는 예시
```swift
// 숫자를 역순으로 정렬하는 함수
for i in (1...10).reversed() {

}

// 숫자를 특정간격으로 범위를 생성하는 함수
// to: 끝값 미포함
// 0 ~ 12
for i in stride(from: 0, to: 14, by: 2) {
    print(i)
}

// through: 끝값 포함 가능
// 0 ~ 14
for i in stride(from: 0, through: 14, by: 2) {
    print(i)
}
```

### 와일드카드 패턴
```swift
// 1. 반복 상수를 생략할 때
for _ in 0..<5 {

}

// 2. 상수/변수로 저장할 필요 없는 경우
let _ = (1...10).reversed()
```

## while문
- 조건을 만족하는 동안 중괄호 내에서 반복적으로 실행함을 의미합니다.
```swift
// 1. 조건문: 참과 거짓을 판단할 수 있는 문장
while 조건문 {
    // 2. 중괄호 내에 반복적으로 실행할 코드를 배치
    // 3. 조건에 변화를 주는 문장(무한루프 방지를 위해 꼭 필여)
}
```

### repeat-while
- 일단 코드를 한번 실행한 뒤 조건에 따라 반복적으로 실행할 지 여부 확인 후 다시 실행합니다.
```swift
repeat {
    // 2. 중괄호 내에 반복적으로 실행할 코드를 배치
    // 3. 조건에 변화를 주는 문장(무한루프 방지를 위해 꼭 필여)
} while 조건문
```

## for문 vs while문
1) for
- 반복횟수를 미리 알고 있거나 컬렉션, 범위 등을 이용할 때
- 또는 반복 상수를 사용하는 작업을 해야할 때
- 범위, 컬렉션, stride, 문자열

2) while
- 반복횟수가 미리 정해져 있지 않고 조건에 따라 바뀔때

## 제어전송문
1) continue
- 반복문에서 다음 주기로 넘어가서 계속함을 의미합니다.
- continue를 만나면 아래 문장들을 무시하고 다음 싸이클로 넘어갑니다.

2) break
- 반복문을 아예 중지합니다.
- switch문에서는 다른 역할로 쓰입니다.

## 참고: 라벨이 붙은 반복문(labeled statement)
- outer:는 반복문에 이름을 붙이는 라벨이고, break outer는 중첩 반복문에서 안쪽이 아니라 해당 라벨이 붙은 바깥 반복문까지 한 번에 종료시키는 문법입니다.
```swift
outer: for i in 0..<3 {
    for j in 0..<3 {
        break outer
    }
}
```
