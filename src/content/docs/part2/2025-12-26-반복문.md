## for문
- 범위나 컬렉션에서 item을 하나씩 뽑아 중괄호 내에서 순차적으로 반복 실행함을 의미합니다.
```swift
for item in 범위 or 컬렉션 {
    print(item)
}
```

### for문의 반복상수
- 아래 item 변수가 아닌 상수로 정의되는 이유
    - 반복 대상의 값을 복사해서 전달하기 때문
    - 실수로 반복 로직을 망가뜨리는걸 방지
    - swift철학흔 가능하면 let, 꼭 필요할때만 var를 사용하라는 철학이 담김
```swift
/**
범위에서 값이 하나씩 item에 복사된다
let item: Int = 0
let item: Int = 1
let item: Int = 2
*/
for item in 0..<3 {
    print(item)
}
```

### 주의점
- for 문에서 선언하는 변수는 for문 내에서만 사용 가능
- 중괄호 내에서 외부의 변수는 접근 가능하지만 외부에서 중괄호 내의 변수는 접근할 수 없다

### for문에서 주로 사용되는 예시
```swift
// 숫자를 역순으로 정렬하는 함수
for i in (1...10).reversed() {

}

// 숫자를 특정간격으로 범위를 생성하는 함수
// to: 끝값 미포함
// 0 ~ 12
for i in stride(from: 0, to: 14, by: 2) {
    print(i)
}

// through: 끝값 포함 가능
// 0 ~ 14
for i in stride(from: 0, through: 14, by: 2) {
    print(i)
}
```

### 와일드카드 패턴
```swift
// 1. 반복 상수를 생략할 때
for _ in 0..<5 {

}

// 2. 상수/변수로 저장할 필요 없는 경우
let _ = (1...10).reversed()
```
