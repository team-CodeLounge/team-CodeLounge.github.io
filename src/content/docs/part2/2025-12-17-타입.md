## 정수
- Swift에서 부호가 있는 정수(signed), 부호가 없는 정수(unsigned)를 지원합니다.
- 이러한 정수는 UInt8(8bit), Int32(32Bit)같이 C와 비슷한 네이밍 형태를 가집니다.

## 정수 범위
- 각 정수 타입의 min, max 프로퍼티를 통해 각 정수 타입의 최소값과 최대값에 접근할 수 있습니다.
```swift
/**
 Unsigned
 - n비트 → 0 ~ (2ⁿ − 1)

 Signed
 - n비트 → −2ⁿ⁻¹ ~ (2ⁿ⁻¹ − 1)
 
 Unsigned는 비트를 전부 값에 쓰고
 Signed는 절반을 부호로 쓴다
 
 
 - Unsigned 정수는 모든 비트를 값 표현에 사용하므로 0부터 2ⁿ − 1까지의 양수만 표현한다.
 - Signed 정수는 전체 비트 중 1비트를 부호에 사용하므로 전체 경우의 수는 같지만 양수로 표현할 수 있는 범위가 절반으로 줄어든다.
 */

// 범위: 0 ~ (2⁸ − 1)
let minValue8 = UInt8.min   // 0
let maxValue8 = UInt8.max   // 255

// 범위: 0 ~ (2³² − 1)
let minValue32 = UInt32.min // 0
let maxValue32 = UInt32.max // 4294967295

// 범위: 0 ~ (2⁶⁴ − 1)
let minValue64 = UInt64.min // 0
let maxValue64 = UInt64.max // 18446744073709551615

// 범위: −2⁶³ ~ (2⁶³ − 1)
// 32-bit 플랫폼에서 Int32와 같은 크기를 가짐
// 64-bit 플랫폼에서 Int64와 같은 크기를 가짐
let intValueMin = Int.min   // -9223372036854775808
let intValueMax = Int.max   // 9223372036854775807
```

## 실수
- 지수와 가수를 나눠서 복잡한 알고리즘으로 저장합니다.
- Float(4 byte): 소수점 6자리까지 정보를 정확하게 저장합니다.
- Double(8 byte): 소수점 15자리까지 정보를 정확하게 저장합니다.
```swift
print("표현 가능한 가장 큰 유한 값 :", Float.greatestFiniteMagnitude) // 3.4028235e+38
print("정규화된 값 중 가장 작은 양수 :", Float.leastNormalMagnitude)  // 1.1754944e-38
print("0이 아닌 값 중 가장 작은 양수 :", Float.leastNonzeroMagnitude) // 1e-45
print("양의 무한대 :", Float.infinity)                               // inf
print("음의 무한대 :", -Float.infinity)                              // -inf
print("숫자가 아님(NaN) :", Float.nan)                               // nan
print("1.0에서의 최소 표현 단위(ULP) :", Float.ulpOfOne)             // 1.1920929e-07


print("표현 가능한 가장 큰 유한 값 :", Double.greatestFiniteMagnitude) // 1.7976931348623157e+308
print("정규화된 값 중 가장 작은 양수 :", Double.leastNormalMagnitude)  // 2.2250738585072014e-308
print("0이 아닌 값 중 가장 작은 양수 :", Double.leastNonzeroMagnitude) // 5e-324
print("양의 무한대 :", Double.infinity)                                // inf
print("음의 무한대 :", -Double.infinity)                               // -inf
print("숫자가 아님(NaN) :", Double.nan)                                // nan
print("1.0에서의 최소 표현 단위(ULP) :", Double.ulpOfOne)              // 2.220446049250313e-16
```

## 변수 타입 문법
- 타입 주석: 타입을 명시적으로 선언합니다.
- 타입 추론: 타입을 명시하지 않고 컴파일러가 타입을 추론합니다.
- 타입 애일리어스: 기존 선언되어 있는 타입, 또는 커스텀 타입에 새로운 별명을 붙여 가독성을 높이는 방법입니다.
